## Cross-cutting themes (multi-lens convergence)

These findings hit independently across multiple agents, signaling real underlying issues:

1. **LLM subprocess egress of unredacted excerpts** (SEC-3 + P1-COUPLING-02). Decision #3's "no excerpt redaction" rationale was scoped to on-disk; LLM mode transmits to Anthropic API before any audit. **Single most-impactful finding overall.**
2. **Layer 2 atomicity / detect-only nature** (FAIL-04 + FAIL-10 + P1-TESTS-01). Layer 2 has a documented two-write window, ships secrets to disk on detection, has no internal-correctness test, and is suppressible by `--quiet`. The architectural backstop has gaps the spec didn't trace.
3. **Storage seam not formalized** (P1-COUPLING-03 + FAIL-03 + SEC-12 + SEC-13). Phase 1.5 deferral rests on a "seam preserved" promise without an interface contract; defaulting `--no-storage` to false multiplies the leak surface.
4. **Decision #3 scope creep** (SEC-3, SEC-4, SEC-12, SEC-14, SEC-1). The "excerpts not redacted" decision has implications for LLM egress, .gitignore defaults, .md sharing, SQLite duplication, and even unrelated fields the rationale didn't sweep.
5. **Test coverage names criteria but not matrices** (P1-TESTS-01 through P1-TESTS-06). Several §13 criteria say "test matrix per code" / "synthetic failure scenarios" without enumerating cases — risk that criteria pass while real surface is uncovered.
